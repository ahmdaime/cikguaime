## ‚ö†Ô∏è Common Errors

### ERROR 1: Lupa Comma
```javascript
‚ùå SALAH:
{ type: "mcq", question: "A?", choices: ["1"], correct: 0 }
{ type: "mcq", question: "B?", choices: ["1"], correct: 0 }

‚úÖ BETUL:
{ type: "mcq", question: "A?", choices: ["1"], correct: 0 },  ‚Üê Ada comma
{ type: "mcq", question: "B?", choices: ["1"], correct: 0 }   ‚Üê Soalan terakhir tak perlu
```

### ERROR 2: Index Salah
```javascript
‚ùå SALAH:
choices: ["A", "B", "C", "D"],  // 4 pilihan (index 0-3)
correct: 4  ‚Üê Index 4 tak wujud!

‚úÖ BETUL:
choices: ["A", "B", "C", "D"],  // 4 pilihan
correct: 3  ‚Üê Index 3 = Pilihan D
```

### ERROR 3: Lupa hasItems
```javascript
‚ùå SALAH:
{ type: "mcq", question: "...", items: ["I", "II"], choices: ["I dan II"], correct: 0 }

‚úÖ BETUL:
{ type: "mcq", question: "...", items: ["I", "II"], choices: ["I dan II"], correct: 0, hasItems: true }
```
